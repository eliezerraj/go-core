# go-core

A reusable Go module with commonly used tools.
- coreJson
    - Read JSON
    - Write JSON
- database
    - Acquire
    - Release
    - GetConnection
    - CloseConnection
    - StartTx
- middleware
    - MiddleWareHandlerHeader
    - MiddleWareErrorHandler
- middleware
    - Event
    - Span

## Installation

go get -u github.com/eliezerraj/go-core
